# DevOps Lab

A collection of hands-on projects covering modern DevOps tools and practices: containerization, CI/CD, Kubernetes orchestration, and cloud deployment.

## Projects

| Project | Technologies | Description |
|---------|-------------|-------------|
| [basic-voting-app](./docker-docker-compose/basic-voting-app/) | Docker, Nginx | Static voting app containerized with Nginx |
| [demo-student-app](./docker-docker-compose/demo-student-app/) | Spring Boot, Docker, GitLab CI/CD, Kubernetes, Helm | Full DevOps pipeline for a Java REST API — from CI to Kubernetes deployment |

## Skills Covered

- **Containerization** — Multi-stage Dockerfiles, image optimization
- **Orchestration** — Docker Compose (dev & prod), Kubernetes (Deployment, Service, Ingress, PVC, Secret)
- **CI/CD** — GitLab pipeline (build → test → package → deploy), Docker Hub publishing
- **Infrastructure as Code** — Kubernetes manifests, Helm Charts
- **Best Practices** — Nginx security headers, environment variables, healthchecks, non-root user
